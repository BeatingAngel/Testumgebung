Teacher-GUI





Options:

Patrol Mode:
If the button is clicked the list of students will be run through. Each student will be shown for 5 seconds before the next student is shown.

Handout:
A ".zip"-file which includes e.g. a list of tasks for the students to complete for the test.

Location:
In the selected path are going to be 3 directories created.
1. Directory: screenshots
In this directory are the screenshots of the students. Each student has his own directory with his screenshots.
2. Directory: submissions
In this directory are the submissions (finished tests) of each student.
3. Directory: exports
In this directory are the exported JAR-Files, Diagrams, . . .

Random-Time:
If the random-time-switch is ON then the screenshots will be created at a random time between 1 second and 30 seconds.

Filename extension:
Specialises the files in which the lines will be counted.
For example: *.java (only the lines in .java-files will be counted. Other files will be ignored.)

Student-Info:

This tab shown all the information of the student.
In the bottom is a graph located which shows how much lines the student has written.
If the students log out during the test, the graph will show a gap in that time interval.
